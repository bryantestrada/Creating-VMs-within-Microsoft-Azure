<h1>Microsoft Azure- Creating Virtual Machines (Computers and Servers)</h1>
This tutorial outlines the creation of virtual computers and servers on the same network using a cloud service provider (Microsoft Azure).<br />

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)
- Windows Server 2022</b>

<h2>List of Prerequisites</h2>

- Microsoft Azure subscription
